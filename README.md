# rock_paper_scissors
<P>This is a simple and interactive Stone Paper Scissors (Rock Paper Scissors) game built using HTML, CSS, and JavaScript. The user plays against the computer, which generates random choices. The game includes score tracking, result display, and dynamic UI feedback for each round.</P>

<h3>Features</h3>
<h4>Real-time gameplay against computer</h4>

<h4>Score tracking for user and computer</h4>

<h4>Visual feedback for win, lose, and draw</h4>

<h4>Responsive design with simple animations</h4>

<h3>How to Run</h3>
<h4>Clone the repository or download the files.</h4>

<h4>Open index.html in any web browser.</h4>

<h4>Play and enjoy!</h4>
